Lógica de Programação JavaScript 03.02 - Exercício 02

Exercício
Para este exercício, mostre na tela todos os anos a partir de 1004 até o ano 2017, pulando de quatro em quatro anos. Por exemplo: 1004, 1008, 1012, etc... É como mostrar os anos bissextos, mas sem as regras para descobrir se um ano é bissexto ou não. (vamos imaginar que se o ano é múltiplo de quatro, ele é bissexto).
