Lógica de Programação JavaScript 06.03 - Exercício

Array de objetos.
Chegou a hora de praticar. Para este exercício, crie um novo documento chamado 06.03_exercicio.html a partir do base. Sua missão será criar um array com três objetos. Cada objeto deverá ter duas propriedades (nome e email). Depois, basta mostrar na tela (melhor se for utilizando a div com id="output") e nome e o email de cada um dos objetos da array criada acima.
